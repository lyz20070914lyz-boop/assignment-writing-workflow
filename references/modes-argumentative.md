# Essay 与 Literature Review

## 共同范例规则

同时比较教师范例和外部可靠范例。外部范例需匹配任务类型、学科、目的和受众，并能看到足够正文。搜索摘要、essay mill、来源不明或 AI 生成样本不能作为主要范例。

## Essay

范例重点：thesis 位置和精度、论证顺序、段落功能、因素整合、反方处理、证据与论点关系、introduction 和 conclusion 功能。

写作逻辑：直接回答题目。每段承担一个论证功能。相关因素需要共同解释时不得割裂。文献用于支持、比较、限制或解释论点，不按研究逐篇总结。

默认顺序：暂定 thesis → 主体论证 → 反方或限制 → introduction → conclusion。

## Literature Review

先判断 narrative、integrative、scoping、systematic 或 critical review。不同类型不能互相套用方法。

范例重点：组织主题、纳入边界、跨研究综合、方法和质量比较、矛盾、研究缺口。

写作逻辑：按主题、机制、方法、争议或时间组织。每个单位比较多项研究，区分一致结果、分歧、方法限制和未解决问题。除非 rubric 要求，不按一篇文章一段组织。

默认顺序：范围与组织框架 → 综合部分 → 分歧与限制 → 缺口或意义 → introduction → conclusion。
