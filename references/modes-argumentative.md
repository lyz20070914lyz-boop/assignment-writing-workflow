# Essay 与 Literature Review

## 范例选择

比较教师范例和可见足够正文的外部可靠范例。外部范例必须匹配任务类型、学科、目的和受众。搜索摘要、essay mill、来源不明或 AI 生成样本不能作为主要范例。

## Essay

使用 thesis 统领全文。检查 thesis 是否直接回答题目、可辩论、范围具体、证据可支持，并能解释各 section 的必要性。

范例重点：thesis 的位置和精度、论证顺序、段落功能、因素整合、反方处理、证据后的推理、introduction 的承诺和 conclusion 的回答。

写作逻辑：每个 section 推进 thesis 的一个必要部分。需要共同解释的因素应在同一因果、条件或比较框架中整合。只处理有证据基础且影响结论的反方或限制。

默认顺序：暂定 thesis → 主体论证 → 反方或限制 → introduction → conclusion。大纲必须显示完整论证链和 rubric 映射。

## Literature Review

先根据题目和课程要求判断 narrative、integrative、scoping、systematic 或 critical review；不确定时询问，不混用报告规范。

使用综合问题、范围和组织主线统领全文。范例重点：纳入边界、主题组织、跨研究综合、方法和质量比较、矛盾、限制和研究缺口。

写作逻辑：按主题、机制、方法、争议或时间组织。每个综合单位比较多项研究，区分一致结果、分歧、方法差异、适用边界和未解决问题。研究缺口必须由已审阅文献支持，不能仅因用户没有找到研究而宣称存在。

默认顺序：范围与组织框架 → 综合部分 → 分歧与限制 → 缺口或意义 → introduction → conclusion。除非 rubric 要求，不按一篇文章一段组织。
